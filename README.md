# Ai
A farsi AI which has developed by Revisto team.

##How to Use it?
###Essentials:
```
git clone "https://github.com/Revisto/Ai"
cd Ai
pip3 install -r Lib/requirements.txt
```
###Run The Ai:
```
python3 Ai.py
```


##Troubleshooting
Having prouble with pyaudio and it records nothing 
EASY, run
```
sudo apt-get install python3-pyaudio
```
